# Changelog

## 2026-02-02
- Se renombro el proyecto a Dito Editor.
- Se agrego menu contextual en el explorador con acciones de documentos y carpetas.
- El selector de workspace usa Finder con soporte de reveal in finder.
- Busqueda del explorer ahora es inline y no abre dialogos del sistema.
- Nuevos documentos se reflejan en el explorador y soportan copiar/pegar.
- Ajustes de pruebas Cypress para asegurar flujos completos.
- Se agrego el script `cy:run:dev` para levantar el renderer y correr Cypress.
- Se habilito seleccionar workspaces desde cualquier unidad.
